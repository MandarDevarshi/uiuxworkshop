In brief, this 5-day workshop is for designers (specifically) and developers (tangentially) who create web UI/UX and wish to

  1. Better their communication between design and dev teams
  2. Create production quality design systems using Adobe Figma
  3. Learn about standards, best practices and current research
  4. Provide documented systems to clients for easy handover, maintenance and future updates
  5. Overall, to manage projects as a freelancer/team/team member efficiently in the long run

For more details check out the section Day wise workshop agenda.

The workshop is aimed at beginners and intermediate skilled would be professionals and professionals.

For past few months I have been pondering and working on combining lessons learned in my art practice with design, management and development of HCIs ( Human-Computer Interfaces ) with a focus on the web. While sharing my thoughts with design and development firms, and hearing them share their issues made me realize there might be a need that this workshop fulfills. So was this workshop born.

There would be some talk about the below mentioned as they are to help in bridging the communication gap which is part of our main agenda but not the main focus of the workshop.

  1. Ideation and Research phase of a project
    1. The workshop is mutually exclusive from that process since that phase is without any bounds and is iterative over an project's life-cycle. Nonetheless we will touch these topics in the beginning and at the end of the workshop I will be sharing resources for learning them.
  2. Javascript language and extended frameworks like React, Angular etc
  3. Component Libraries like Bootstrap, Material UI etc
  4. Markup and styling in HTML, Cascading Style Sheets (CSS)
  5. Browser Developer Tools in entirety, only parts relevant to workshop and designers will be covered
